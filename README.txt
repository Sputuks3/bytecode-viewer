Bytecode Viewer (https://bytecodeviewer.com/) is an Advanced Lightweight Java Bytecode Viewer, GUI Java Decompiler, GUI Bytecode Editor, GUI Smali, GUI Baksmali, GUI APK Editor, GUI Dex Editor, GUI APK Decompiler, GUI DEX Decompiler, GUI Procyon Java Decompiler, GUI Krakatau, GUI CFR Java Decompiler, GUI FernFlower Java Decompiler, GUI DEX2Jar, GUI Jar2DEX, GUI Jar-Jar, Hex Viewer, Code Searcher, Debugger and more.
It's written completely in Java, and it's open sourced. It's currently being maintained and developed by Konloch.

Check it out here (https://github.com/Konloch/bytecode-viewer)

This is Brendan's modified version of BCV, aimed at supporting the JEB decompiler.

8 May, 15 - With added support for JEB decompiler (https://www.pnfsoftware.com/)
	- JEB Decompiler is a commercial decompiler for Android and Dalvik executables. We are unable to include this in 	our project, so the best approach was to run it in a separate process, with the necessary parameters.
